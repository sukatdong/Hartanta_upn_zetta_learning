Angular adalah framework javascript yang memungkinkan untuk membuat Single-Page-Applications (SPAs) reaktif. 
kerangka pemrograman struktural untuk mendapatkan web app yang dinamis.
Hal ini memungkinkan menggunakan HTML sebagai bahasa template serta memperluas sintaks HTML 
sehingga seluruh komponen dalam aplikasi yang tengah dibangun bisa diekspresikan secara jelas dan ringkas.
angular hanya menggunakan satu file HTML dan beberapa Typescript. 


Module adalah sekumpulan komponen yang didedikasikan untuk domain aplikasi, alur kerja, atau sekumpulan kemampuan yang terkait erat.

Component adalah bagian-bagian kecil dari keselurahan aplikasi seperti UI.

Hubungan antara component dan module adalah Modulediisi oleh kelompok component yang saling berhubungan. 
sementara component membutuhkan module untuk mengidentifikasi component lain yang memiliki hubungan yang sama.

KUIZZ 
1. Component dari tampilan itu dibagi menjadi html, typescript dan SCSS.
Html menjadi ouput/tampilannya sedangkan typescript dan SCSS merupakan isi dari Html nya

2. Component akan dikelompokkan sesuai dengan modulnya, contohnya untuk musik akan menampilkan komponen musik. 

3. Projek angular memuat modul yang diisi oleh file html jadi dari HTML tersebut kita bisa melihat bentukan dari projeknya
